# Science 4 Humans Website

Welcome to your first website project! 🎉

## Project Structure

- `index.html` - Your main HTML file with the basic structure
- `styles.css` - Your CSS file for styling (includes the color palette)
- `script.js` - Your JavaScript file for interactivity
- `README.md` - This file

## Color Palette

The colors are already set up in `styles.css` as CSS variables:
- `--blue`: Deep ocean blue (#1e3a5f)
- `--green`: Sea green (#2d5a4e)
- `--white`: White (#ffffff)
- `--sandy-tan`: Sandy tan (#d4c5a9)

You can use them like this: `color: var(--blue);`

## Sections to Build

1. **Navigation** - Menu bar at the top
2. **Hero Section** - Main banner/intro
3. **Our Purpose** - About the organization
4. **Events** - Black History Month, Women's Month, Food Drive
5. **Mini Events** - Mini Libraries
6. **Previous Campaigns** - Past work
7. **News** - Latest updates
8. **Blog** - Informational posts
9. **Partnering with Us** - Collaboration section with donation button
10. **Footer** - Contact info, links, etc.

## Getting Started

1. Open `index.html` in your browser to see your website
2. Edit the HTML file to add content
3. Style it with CSS in `styles.css`
4. Add interactivity with JavaScript in `script.js`

## Tips

- Take it one section at a time
- Don't worry about making it perfect right away - you can always improve it later
- Experiment and have fun!
- Use browser developer tools (F12) to test and debug

Good luck! You've got this! 💪
